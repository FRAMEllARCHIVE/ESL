Key Phrases for Shopping

Now let's learn some key phrases that you can use when you go shopping!

How much is this? - Use this phrase to ask for the price of an item. Example: "How much is this notebook?"

Can I help you? - A common question a shopkeeper might ask. Example: "Can I help you find something?"

I would like to buy... - Use this phrase when you want to say what you want to purchase. Example: "I would like to buy a pen."

Where can I find... - Use this phrase to ask where an item is in the store. Example: "Where can I find the snacks?"

Do you have... - Use this phrase to ask if a shop has a particular item. Example: "Do you have any water bottles?"

I’ll take it! - Use this phrase when you decide to buy something. Example: "I’ll take this pencil!"
